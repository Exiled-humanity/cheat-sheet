Basic Structure

Tag Name Purpose
<!DOCTYPE HTML> Doctype HTML Sets page to html
<html> html Needed to code html elements
<head> head A section to encode meta-data
<body> body Section for the main body of the page
<h1> heading main heading for the page
<h2 - h6> Smaller headings sub-headings for the page
<p> Paragraph A tag for inputting paragraphs
<div> Div A tag for id/styling between elements
<span> Span A tag for id/styling within elements
<br> Break inserts a line break

Semantic Structure

Tag Name Purpose
<nav> Navigation Container for navigational links
<main> Main Container for main section of the page
<article> Article Container to signal article content
<aside> Aside Container for additional information
<footer> Footer Container for footer

Formatting Tags

Tag Name Purpose
<blockquote> blockquote To add a blockquot to the page
<q> Quotation Marks To control quote marks in a blockquote
<blockquote cite> Citiation Attribution to add citation source/url
<strong> Strong Creates an important (bold) statement
<em> Emphasis Creates a stressed (italics) statemen
<small> Small Creates a smaller font
<s> Strike-out Create strike-out text
<u> underline Creates an underlined text
<pre> Preformatted Creates performatted text
<sup> Superscript To insert superscript text
<sub> Subscript To create subscript
<a> Anchor text URL links, Fragment Links or tooltips

List Tags

Tag Name Purpose
<ul> Unordered list To create a bulleted list
<ol> Ordered List To create an ordered list
<="list-style-type:"> List style type Used to code a specific bullet/list type
<circle> Circle Used with list-style-type= to create circle bullets on ULs
<square> Square Used with list-style-type to create square bulllets in UL
<URL(addy.file)> Used to create custom image bullets sub-headings for the page
<upper-roman> Uppercase Roman Numerals Changes numbers to upcase roman numerals in ordered lists
<start> Starting point Creates a starting point for the list eg 100
<dl> Description list Used for list with descriptions eg question answers/definitions
<dt> Definition term The question/definition in a DL
<dd> Details tag Answer or definition in a DL
<dfn> Definition tag To signal a definition in a DL

Table Tags

Tag Name Purpose
<table> Table Creaes a table
<tr> Table Row Creates a table row
<td> Table Data Creates a data cell
<caption> Caption Creates a table caption
<rowspan="2"> Row Span Creates td that spans multiple rows
<colspan="2"> Column span Creates a td that spans multiple columns
<thead> Table Header Tag for a table header
<tbody> Table Body Tag for a table body
<tfooter> Table Footer Tag for a table footer
<br> Break inserts a line break

Media Tags

Tag Name Purpose
<img src="url"> Image Inserts an image
<map> Image map Creates an image map
<coords> Coordinates: Define coordinates on a map
<alt> Alt text Creates alt text on image
<figure> Figure Figure text to embed images/add ids
<figcaption> Figure Caption Add a cpation to images
<iframe src=""> iframe Fixed area on page for an embedded media
<audio src=""> Audio file Embed an audio file
<video src=""> Video Embed a video
<controls> Controls Control attribute for media functions
